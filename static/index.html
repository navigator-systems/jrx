Welcome to JRX

JRX is a powerful project generator that helps you quickly scaffold new projects from templates. Whether you're starting a new web application, CLI tool, or microservice, JRX provides the templates and tools you need to get started fast.

Use the navigation menu on the left to explore templates, create new projects, configure your settings, and access the API. Each section provides intuitive operations similar to the CLI experience, but with a modern web interface.
